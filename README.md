# Projeto-jogo-Pong-em-Java
Projeto de um jogo estilo Pong desenvolvido utilizando Java e HTML.

## Sobre o projeto
Este projeto foi criado com o objetivo de praticar conceitos de lógica de programação, incluindo:

- Manipulação de elementos na tela
- Lógica de jogo
- Interação com o usuário

(Este é um projeto antigo e pode não estar totalmente funcional ou otimizado.)

## Tecnologias utilizadas

- JavaScript
- HTML
- CSS

## Funcionalidades

- Movimentação das barras
- Bola em movimento
- Sistema básico de colisão

## Como executar
abrir o arquivo 'index.html' em seu navegador.
